## 📡 Core Networking Commands

### 🔹 ping
Used to test connectivity between devices and verify network reachability.

### 🔹 traceroute / tracert
Shows the path packets take to reach a destination and identifies latency points.

### 🔹 netstat
Displays active connections, listening ports, and protocol statistics.

### 🔹 arp -a
Displays the ARP table and maps IP addresses to MAC addresses.

### 🔹 ipconfig / ifconfig
Shows IP configuration details of a device.
